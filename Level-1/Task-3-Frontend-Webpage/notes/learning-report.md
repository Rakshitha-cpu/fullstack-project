# 🚀 Task 3 - Frontend Webpage Learning Report

---

## 📌 1. Project Overview
This project is a frontend web application built using **HTML, CSS, and JavaScript**.  
It focuses on understanding how a real-world web interface is structured, styled, and made interactive.

The main goal was to build a functional UI and connect basic frontend logic using JavaScript.

---

## 🎯 2. Objective
The objective of this task was to:

- Build a structured frontend webpage
- Understand HTML layout design
- Apply CSS for styling and responsiveness
- Use JavaScript for basic interactivity
- Organize project files in a proper structure

---

## 🛠️ 3. Tech Stack

| Technology | Purpose |
|------------|--------|
| HTML5      | Page structure |
| CSS3       | Styling and layout |
| JavaScript | Interactivity and DOM manipulation |

---

## 💡 4. Features Implemented

- Dashboard-style UI layout
- Display of user data on screen
- Search functionality for filtering data
- Interactive elements using JavaScript
- Clean and modular file structure

---

## 📚 5. Key Learnings

Through this project, I learned:

- How HTML builds the structure of a webpage
- How CSS controls design, spacing, alignment, and layout
- How JavaScript manipulates DOM elements dynamically
- How frontend components communicate with each other
- Importance of proper project organization
- How small UI projects are structured in real development

---

## ⚠️ 6. Challenges Faced

- Understanding proper file linking between HTML, CSS, and JS
- Debugging JavaScript logic issues
- Managing folder structure correctly
- Handling placement of assets like images and screenshots
