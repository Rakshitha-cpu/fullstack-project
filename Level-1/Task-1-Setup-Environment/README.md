# 🚀 Level 1 - Task 1: Setup Development Environment

## 📌 Project Overview

This task focuses on setting up a professional full-stack development environment required for modern web application development. The environment includes programming tools, version control systems, package managers, and development utilities used by software engineers in industry.

---

## 🎯 Objectives

* Install and configure Node.js and npm.
* Set up Git and GitHub for version control.
* Configure Visual Studio Code as the primary IDE.
* Learn and practice essential terminal commands.
* Prepare the system for future full-stack development tasks.
