## Problems Faced

### Git Push Error

**Issue:**

error: src refspec main does not match any

**Cause:**

Attempted to push code before creating the first Git commit.

**Solution:**

Created an initial commit using Git and then pushed the repository successfully.

### Command Line Confusion

**Issue:**

Confusion between `cd` and `mkdir` commands.

**Solution:**

Practiced basic terminal navigation and folder creation commands.
